# Public Release Notes

## Who this is for
- OpenClaw users who live inside Telegram
- users who want a mobile-friendly operational board
- users who do not want a full browser dashboard for every quick check

## What this skill does
- defines a compact Telegram status board format
- provides a renderer script
- provides a basic OpenClaw collector script
- makes the board easy to reuse and extend

## What it does not do yet
- it does not directly send Telegram messages by itself
- it does not replace a full dashboard
- it does not yet provide deep cost analytics
